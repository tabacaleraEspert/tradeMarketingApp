
  # Trade Marketing Mobile App

  This is a code bundle for Trade Marketing Mobile App. The original project is available at https://www.figma.com/design/2Ccx0bLGD2iasyfqAX09Sv/Trade-Marketing-Mobile-App.

  ## Running the code

  ```bash
  npm install    # Instalar dependencias
  npm run dev    # Iniciar servidor de desarrollo (http://localhost:5173)
  # o
  npm start      # Alternativa para iniciar
  ```

  La app se abre automáticamente en http://localhost:5173
